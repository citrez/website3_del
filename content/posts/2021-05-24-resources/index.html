Here is a collection of things that I love to use. I hope to update this regularly

R and packages

  • R for Data Science for me is the definitive guide to Data Science in R and can help you think about data problems. It is both highly approachable and will really get you quite far along with your data needs.

  • The resource for building package in R. R Packages by Hadley Wickham.

  • I used hugo and blogdown to create this site. The blogdown book is the best place to start and this blog is a nice primer.

  • Tidytext for interpretable ways of gaining insight from text data

Bloggers

  • Mike Dane’s you tube channel is very good. I used him to learn about hugo, which was used to build this site, but there is a lot more on there.

  • Alison Hill is a recent find and an absoloute teasure strove of information.

Cedric Scherer, what a man, what a blog.

  • Julia Silge has an incredible youtube channel, taking you through all things tidymodeling. Super useful resource, if, like me, you’d like a framework to rival scikit-learn that will keep you safely within the confines of Rstudio. She also has a blog

  • Ryans tutorials are a cool resource for introductionary lesson to some programming fundementals

  • Story Bench has tons of interesting bloggs

  • StatQuest with Josh Stalmer provides simple explanations of all things Stats/ML and comes with a healthy dose of ironic uekele playing, whats not to like?! DOIBLE BAMM.

-fireship for all my interest in webdev. Offers great youtube 100 second snippets.

Visualising

  • The R Graph gallery and this website as a whole is an absolutely indispensable resource. Not only does it have practically every visualization that you could possibly hope to make in r, but it gives you simple reproducible code to go along with it. This site similarly caters for python, and D3 (would love to learn more D3, so watch this space). It also links to their award winning data-to-vis websites, which offers a decision tree (under the ‘explore’ tab) to pick appropriate visual along with the relevant caveats.

  • GGplot2 book by hadley is the definitive guide for ggplot2, which is the go-to vis package in R. I always like to invest in a hardcopy, more likely to actually read it.

  • Cedrics ggplot2 tutrial is a great place to go if you’re looking to be inspred. Or infact any/all of his visualisations

Other great books/sites

  • For learning markdown this is great interactive tutrial.

  • An introduction to statisical learning with applications in R. A great book for understanding the key ideas in ML, offers a readable and no-nonsense introduction. Although I find the “application in R” a bit outdataed ‘base R’-y for my liking, the funenmental concepts have stood the test of time.

  • Regex is like a superpower. Once you know it, you’ll wonder how you ever got by without it. The good news is, it is more important to know what is possible, that remember exactly how to implement it

Amazing Websites that deserve more attention

  • Have you ever wished you know more about git? (the answer should always be yes) git-school have created this neat little webpage that helps you visualise what goes on when you hit that dreaded git pull.

  • CSS selectors, useful for telling a computer which part of a webpage you want it to select flukeout does an incredible job at game-ifying CSS selection. If you’re not careful, you won’t even realise you’re learning.

  • Library of Babel makes you think about randomness

  • Pudding is website dedicated to cool visualization. Worth checking out for some inspo.

Less-well-known Packages I dig

  • SkimR - provide summary statistics about variables in data frames
